# MCP Server Guide

A comprehensive visual guide for building custom MCP (Model Context Protocol) servers with TypeScript.

[https://mauriciobarriosb.github.io/mpc-server](https://mauriciobarriosb.github.io/mpc-server)

---

## About

This project provides detailed documentation covering:

- **Execution Flow** - Step-by-step communication flow between Claude and MCP servers
- **STDIO Architecture** - How stdin/stdout transport works
- **Server Anatomy** - Core components and structure of an MCP server
- **Configuration** - Setting up `claude_desktop_config.json`
- **JSON-RPC Protocol** - Message format and request/response patterns
- **Debugging** - Error handling and troubleshooting techniques
- **Comparison** - MCP vs traditional APIs
- **Project Setup** - Getting started with your own MCP server
- **Advanced Tools** - Building complex tool implementations
- **Resources** - Dynamic data exposure patterns
- **Prompts** - Reusable prompt templates
- **Real-world Examples** - Practical use cases
- **Best Practices** - Production-ready patterns
- **Advanced Patterns** - Middleware, caching, and more

## Tech Stack

- React 19
- TypeScript
- Vite

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
